
1. creating a pin
    - the user goes to the landing page
    - they should login with google email
    - the user grants location permission
    - the user then rerouted to the map page
    - the user hits the add pin button(click point on map)
    - the user is then prompted for time place details and images
        - time and palce can be automatically determined
        - images are optional
    - the user then hits submit
    - all nearby (by zipcode?) users are alerted that there's been a new sighting

2. Check alerts
    - the user goes to the landing page
    - they should login with google email
    - the user grants location permission
    - the user then rerouted to the map page
    - the user can click on a pin
    - a popup appears with the details of the sighting
    - the user can upvote or downvote
